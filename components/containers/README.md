# Containers

Boxes, panels and cards that hold a number, an idea, or a comparison.

- `kpi-tile/` &mdash; one loud number with a short descriptor, optionally an icon.
- `callout/` &mdash; a labelled box: coloured-header info box, or a dashed assumption note.
- `takeaway-strip/` &mdash; a full-width navy (or activist black) bar closing out an exhibit.
- `comparison-card/` &mdash; a card in a peer-comparison grid, accent top-border on the hero.

See each component's `spec.md` for anatomy and reskinning notes.
