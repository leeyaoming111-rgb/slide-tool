# Tables

Rows and columns, horizontal rules only: no outer border, no vertical lines.

- `stat-table/` &mdash; a clean two-column label/value list.
- `scenario-table/` &mdash; bear / base / bull table with fixed red/blue/green headers.

See each component's `spec.md`, and `../../docs/CHART_RULES.md` rules 11 and 20 for the table
conventions (right-aligned numerics, horizontal rules / row banding only).
